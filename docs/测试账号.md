## 如何使用

1. **自动导入**（推荐）
   - 启动后端应用时，Flyway会自动执行 `V3__Insert_sample_data.sql`
   - 示例数据会自动插入数据库

2. **手动执行**（如需重置）
   ```sql
   -- 1. 清空现有数据
   DELETE FROM notification;
   DELETE FROM like_record;
   DELETE FROM comment;
   DELETE FROM post;
   DELETE FROM user;

   -- 2. 然后执行 V3__Insert_sample_data.sql 中的INSERT语句
   ```

3. **登录测试**
   - 访问 http://localhost:3001/login
   - 使用上述任一账号登录（密码：password123）
   - 即可看到完整的示例数据和交互

## 清空测试数据

如果需要清空所有测试数据重新开始：

```sql
-- 按依赖顺序删除
DELETE FROM notification WHERE user_id <= 5;
DELETE FROM like_record WHERE user_id <= 5;
DELETE FROM comment WHERE user_id <= 5;
DELETE FROM post WHERE user_id <= 5;
DELETE FROM user WHERE id <= 5;

-- 重置自增ID（可选）
ALTER TABLE user AUTO_INCREMENT = 1;
ALTER TABLE post AUTO_INCREMENT = 1;
ALTER TABLE comment AUTO_INCREMENT = 1;
ALTER TABLE like_record AUTO_INCREMENT = 1;
ALTER TABLE notification AUTO_INCREMENT = 1;
```
