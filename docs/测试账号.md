## 示例数据说明

系统会通过 Flyway 自动导入示例数据。

- 全新数据库首次启动时，会自动执行 `V3__Insert_sample_data.sql`
- 如果你已经保留了 MySQL 数据卷，那么示例数据通常已经存在，不需要重复导入

## 如何使用

### 1. 登录测试

当前前端入口地址：

- http://localhost:3000/login

使用上述任一账号登录即可。

### 2. 验证后端状态

如果登录前想先确认后端是通的，可以访问：

- http://localhost:8080/api/test/hello

### 3. 重置测试数据

如果你希望回到一套全新的测试数据，推荐直接重建 Docker 卷：

```powershell
cd D:\EmotionHub
docker compose down -v
docker compose up -d --build
```

不建议把 `database/init.sql` 当作日常重置方式。当前主流程已经迁移到 Flyway。
