## 🔑 配置说明

### 通义千问API配置

在 `application-dev.yml` 中配置：

```yaml
dashscope:
  api-key: ${DASHSCOPE_API_KEY:}  # 环境变量或直接填写

emotion:
  analysis:
    provider: qwen  # keyword(默认) 或 qwen
```

或设置环境变量：
```bash
export DASHSCOPE_API_KEY=your_api_key_here
```

### 启动项目

**后端：**
```bash
# 1. 启动数据库
docker-compose up -d

# 2. 启动后端（在backend/emotion-hub-web目录）
mvn spring-boot:run
```

**前端：**
```bash
# 3. 启动前端（在frontend目录）
npm run dev
```

**访问：**
- 前端：http://localhost:3001
- 后端API：http://localhost:8080/api
- API文档：http://localhost:8080/doc.html
